Click "Menu" to open

This is a faithful reproduction of the Impronta site menu animation. The trigger button on the top-left expands into a full-screen menu via a clip-path animation. As it expands, the trigger's text characters slide up and away. Once open, each section's text reveals character-by-character with a staggered delay.

Hover over any link to see the character-by-character swap effect: the text slides up while a clone slides in from below. Each character is offset by 2.4ms creating a wave-like motion.

The whole text animation system runs purely in CSS — JavaScript only splits the text into spans and toggles classes. No GSAP, no animation libraries.
